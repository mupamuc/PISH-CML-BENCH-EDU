Как участвовать · два пути

Путь 1 — гость (проще всего)

Ничего устанавливать и нигде регистрироваться не нужно.

  1. Погуляйте по платформе: обзор, трекер, кабинеты, глоссарий.
  2. Вернитесь сюда и заполните форму: имя, раздел, идея. Готово — она уже на графе, и её видят все.
  3. Нажимайте на чужие идеи: комментируйте и голосуйте «Поддержать».

Путь 2 — через GitHub (для правок «с реализацией»)

Если готовы не только предлагать, но и вносить изменения в сами страницы. Понадобится бесплатный аккаунт GitHub.

  1. Зарегистрируйтесь на github.com (почта + пароль, 2 минуты).
  2. Откройте страницу проекта и нажмите кнопку Fork справа сверху. Форк — это ваша личная копия: правьте смело, оригинал не пострадает.
  3. В своей копии найдите нужный файл (например, index.html), нажмите на карандаш ✏ и правьте текст прямо в браузере.
  4. Внизу нажмите Commit changes — это «сохранить с подписью, что и зачем изменили».
  5. GitHub сам предложит кнопку Contribute → Open pull request. Pull request — это ваше предложение владельцу проекта. Напишите пару слов, зачем правка, и отправьте.
  6. Дальше видно здесь, в ленте ниже: предложение «ожидает проверки» → «принято в основную версию».
