# Платформа ПИШ ЦИ БАС — демо (фаза 0)

Публичный контур платформы магистратуры «Системный цифровой инжиниринг БАС» (15.04.03, ПИШ СПбПУ).
Самодостаточный статический сайт: без сборки, без внешних зависимостей — открывается локально и на GitHub Pages.

## Страницы
- `index.html` — лендинг программы: позиционирование, учебный план (интерактивный фильтр по модулям/семестрам), сквозной проект, ИИ-агенты, преимущества, набор 2026.
- `tracker.html` — публичный трекер сквозного проекта: команды, стадии С1–С4, прогресс, артефакты, дорожная карта изделия (клик по команде — детали).
- `app.html` — кликабельный прототип кабинетов: студент · преподаватель · РОП · директор.
- `ideas.html` — «Экспертный совет»: живой граф идей и комментариев экспертов (гости пишут без регистрации, хранилище — Supabase, обновление в реальном времени) + лента событий GitHub на человеческом языке + инструкция «как участвовать» (гость / fork → pull request).

## Данные
- `assets/data.js` — реальный учебный план (g15.04.03_15_ПИШ_2026.plx) + модель проекта из аналитики uav-msc-review. Команды и прогресс — синтетические для демонстрации.
- `assets/theme.css` — дизайн-система «инженерная телеметрия».
- `assets/ui.js` — общие компоненты (навигация, футер).

## Участие экспертов
Страница «Идеи» (`ideas.html`) — открытая доска: идеи и критика прикрепляются к разделам платформы и видны всем сразу. Два пути участия: гость (представился — пишет, без регистрации) или GitHub (fork → правка → pull request, инструкция на странице). Подключение хранилища — [docs/SETUP-SUPABASE.md](docs/SETUP-SUPABASE.md).

## Запуск
Открыть `index.html` в браузере. Или деплой на GitHub Pages: Settings → Pages → Deploy from branch → main → /demo (или /root, если положить в корень).

## Что демонстрирует (ключевое из плана)
- Аналитика учебного плана = модель данных платформы (карта тем → граф, дисциплина→артефакт → реестр проекта).
- V&V-чек-лист на приёмке артефакта (кабинет преподавателя): расчёт не закрыть без цепочки достоверности ГОСТ Р 57700 — программно закрывает главный пробел плана.
- ИИ-наставник с «честным контуром»: включает преподаватель, помогает решить, не решает за студента.
- РПД-контур с поиском дублей против карты владения темами (кабинет РОП).
